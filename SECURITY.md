# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Crownan, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email: **kjartan1996@gmail.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and work with you to resolve the issue before any public disclosure.
